# RomChipReader
# 1 Overview 
A tool for read memory written in IP3604 (К556РТ5) and IP3601 (К556РТ4) 
Инструмент для чтения (получения дампа памяти) с микросхем ПЗУ К556РТ4 и К556РТ5
